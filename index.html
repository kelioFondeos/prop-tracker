Cómo configurar:
1. GitHub → Settings → Developer settings → Personal access tokens → New token (classic)
2. Marca solo el scope gist → Generate token
3. Pega el token abajo, deja el Gist ID vacío la primera vez → Guardar
4. Copia el Gist ID generado para usarlo en otros dispositivos
